#description

## erp x0.1

One mini ERP system made for final exam.

gulp + mongodb + express

# How to start

install dependencies:

```
  $ cd . && npm install
```

run the app:

```
  $ DEBUG=erp:* npm start
```

# framework libray

1. [Semantic-ui](https://github.com/Semantic-Org/Semantic-UI)

1. [react 0.13.0](http://reactjs.cn/react/downloads.html)
